Smart ATM Database & Transaction Management System
Overview
You are required to design and develop an ATM Backend & Database System using:
Java 25 (advanced features mandatory)
Spring Boot + gRPC (RPC-based communication)
Gradle (mandatory build tool)
OAuth 2.0 Authentication (Spring Security)
MySQL (Relational Database)
The system simulates a real-world banking ATM network platform handling:
Account management
Cash withdrawal & deposit
Balance inquiry
Fund transfers
ATM machine monitoring
Transaction processing & reconciliation
Fraud detection & compliance
Architecture Requirements
Gradle multi-module architecture
MySQL with strong ACID guarantees
High concurrency transaction handling
Distributed ATM network support
Idempotent financial operations
Security
OAuth 2.0 with JWT

gRPC Contract Example (Proto)
service TransactionService {
 rpc InitiateWithdrawal (WithdrawalRequest) returns (TransactionResponse);
 rpc GetBalance (AccountRequest) returns (BalanceResponse);
}

message WithdrawalRequest {
 string accountId = 1;
 double amount = 2;
}

message TransactionResponse {
 string transactionId = 1;
 string status = 2;
}
